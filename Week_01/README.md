## java字节码技术
> java bytecode有单字节的指令组成，理论上支持256个，实际只使用了200多个左右。  

### 根据指令的性质，主要分为四类：  
1. 栈操作指令
2. 程序流程控制指令
3. 对象操作指令，包括方法操作指令
4. 算数运算以及类型转换指令

### 生成字节码
1. 编译 ： javac Xxx.java  
2. 查看 ： java -c -verbose Xxx.class
